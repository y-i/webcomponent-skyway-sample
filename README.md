# webcomponent-skyway-sample

[SkyWay v2の終了](https://support.skyway.io/hc/ja/articles/40227790620313--%E9%87%8D%E8%A6%81-%E6%97%A7SkyWay-WebRTC-Platform-SkyWay-%E3%81%AE%E3%82%B5%E3%83%BC%E3%83%93%E3%82%B9%E7%B5%82%E4%BA%86%E3%81%AB%E3%81%A4%E3%81%84%E3%81%A6)が近いのでアーカイブ化しました。

SkyWay v2のSFUを利用したWebComponentの例です。
自動で指定したRoomに入室し、最後に追加された映像を出力します。

## 使い方

1. `npm install`
1. `main.ts`内のAPI KeyとRoom名を書き換える
1. `npm run dev`
1. 表示されたURLに複数タブでアクセスし、表示されることを確認
